It's a physical 2D engine made in Python

<h4>Installing</h4>
u need python 3.10+

```pip install pygame```

```pip install pymunk```

<h4>Controls</h4>
LMB - Spawn a square

RMB - Spawn a circle

C - delete ALL objects in map

ESC - exit
